# Installation 
1. Install [Link VimPlug](https://github.com/junegunn/vim-plug)
2. ```mv ~/.config/nvim ~/.config/nvimBackup```
3. ```git clone https://github.com/SIB61/SIB61-VIM ~/.config/nvim```
4. ```nvim +PlugInstall```
