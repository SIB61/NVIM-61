# SIB61-VIM
NeoVim Configuration with important plugins.

# Installation 
1. Install [VimPlug](https://github.com/junegunn/vim-plug)

3. ```mv ~/.config/nvim ~/.config/nvimBackup```

5. ```git clone https://github.com/SIB61/SIB61-VIM ~/.config/nvim```

7. ```nvim +PlugInstall```
